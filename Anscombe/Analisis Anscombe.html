DATASET ANSCOMBE¶

El conjunto de datos de Anscombe (Anscombe's Quartet) es un famoso conjunto de datos que fue creado por el estadstico britnico Francis Anscombe en 1973. El conjunto de datos consiste en cuatro grupos de datos (I, II, III y IV), cada uno con las mismas propiedades estadsticas bsicas, pero con distribuciones y relaciones visuales muy diferentes cuando se grafican.

Gráficas de Cada Grupo¶

In [30]:
import matplotlib.pyplot as plt

# Crear subplots con 4 grficos (uno para cada grupo)
fig, axs = plt.subplots(2, 2, figsize=(12, 10))

# Lista de ejes para facilitar el acceso
ax_list = axs.ravel()

# Graficar cada grupo en su propio subplot
for i, group in enumerate(df['group'].unique()):
    group_data = df[df['group'] == group]
    ax = ax_list[i]
    ax.scatter(group_data['x'], group_data['y'])
    ax.set_title(f'Grupo {group}')
    ax.set_xlabel('x')
    ax.set_ylabel('y')
    ax.grid(True)

# Ajustar el espaciado entre grficos
plt.tight_layout()
plt.show()

A pesar de tener estos valores estadsticos idnticos, cada uno de los conjuntos de datos tiene una distribucin visual muy diferente, lo que resalta la importancia de no depender nicamente de estadsticas descriptivas para analizar datos y, en cambio, visualizar las relaciones entre variables.

Gráfica de Dispersión de x vs y para Todos los Grupos con Línea de Regresión¶

In [31]:
from sklearn.linear_model import LinearRegression

# Crear una sola grfica de dispersin con todos los grupos
plt.figure(figsize=(12, 10))

# Graficar cada grupo en el mismo grfico
for group in df['group'].unique():
    group_data = df[df['group'] == group]
    plt.scatter(group_data['x'], group_data['y'], label=f'Grupo {group}')

# Calcular la regresin lineal usando todos los puntos de los grupos
X = df['x'].values.reshape(-1, 1)  # Variable independiente
Y = df['y'].values  # Variable dependiente

# Crear el modelo de regresin lineal
regression_model = LinearRegression()
regression_model.fit(X, Y)

# Predicciones del modelo
y_pred = regression_model.predict(X)

# Dibujar la lnea de regresin en la grfica
plt.plot(df['x'], y_pred, color='red', linewidth=2, label='Lnea de Regresin')

# Configurar el grfico
plt.xlabel('x')
plt.ylabel('y')
plt.grid(True)
plt.legend()
plt.show()

Conclusión¶

El cuarteto de Anscombe es un excelente ejemplo de la necesidad de visualizar los datos, ya que los valores estadsticos como la media, la varianza y la correlacin no cuentan toda la historia de la distribucin de los datos.

Si se tiene que hacer un anlisis de datos o se tiene que trabajar con datos estadsticos, es importante siempre visualizar los datos adems de calcular las estadsticas, para evitar llegar a conclusiones errneas basadas nicamente en nmeros.
